## Template 5 — Bing & Yandex Webmaster Setup

**Trigger:** Phase 4 Multi-Engine Indexing
**Status:** ⏳ ready to send

```
Hi Kevin, für Bing & Yandex (zusätzlich zu Google Search Console)
sind 2 kostenlose Konten nötig — ich helfe beim Einrichten.

══════════════════════════════════
🔍 BING WEBMASTER TOOLS (5-10 min)
══════════════════════════════════

1. Gehe auf bing.com/webmasters
2. Klicke "Sign in" → mit deiner Microsoft-Email einloggen
   (falls keine: kostenlos auf signup.live.com erstellen)
3. "Add a site" → rundumshaus-littawe.de eingeben
4. Verification: wähle "Google Search Console import" (am einfachsten)
   → es importiert automatisch deine bestätigte Site aus GSC
5. Sitemap submit: rundumshaus-littawe.de/sitemap.xml
6. Schick mir Screenshot vom Dashboard — fertig!

══════════════════════════════════
🔍 YANDEX WEBMASTER (5-10 min, optional)
══════════════════════════════════

1. Gehe auf webmaster.yandex.com
2. Sign up mit Email (oder bestehender Yandex-Account)
3. "Add site" → rundumshaus-littawe.de
4. Verification per HTML meta tag:
   - Yandex zeigt dir einen Code wie:
     <meta name="yandex-verification" content="abc123def..." />
   - Schick mir nur den content-Wert ("abc123def...")
   - Ich baue ihn in die Seite ein, dann kannst du verifyen
5. Submit sitemap

══════════════════════════════════

Beide nicht zwingend, aber Bing fängt 10-15 % der DACH-Suchanfragen
ein — und ChatGPT Search nutzt Bing-Index. Lohnt sich.

Mach das wenn Zeit ist — kein Stress.
```
