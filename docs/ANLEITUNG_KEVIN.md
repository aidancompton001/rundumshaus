# Anleitung: Website-Inhalte bearbeiten

## So geht's

1. Öffne **[app.pagescms.org](https://app.pagescms.org)** in deinem Browser
2. Melde dich mit deinem **GitHub-Konto** an
3. Wähle das Repository **rundumshaus** aus
4. Du siehst jetzt alle Bereiche deiner Website

---

## Was du ändern kannst

### Firmendaten
- Firmenname, Telefon, E-Mail, Adresse
- Klicke auf **Firmendaten** → ändere die Felder → **Save**

### Startseite
- Hero-Text (große Überschrift + Unterüberschrift)
- Über uns (Text und Überschrift)
- Statistik (Zahlen)
- Klicke auf **Startseite** → ändere die Felder → **Save**

### Weitere Dienstleistungen
- Neue Dienstleistung hinzufügen oder bestehende entfernen
- Klicke auf **Weitere Dienstleistungen** → bei "Dienstleistungen" auf **+** klicken → Name eingeben → **Save**

### Hauptleistungen
- Titel und Beschreibungen der 5 Hauptleistungen ändern
- Klicke auf **Hauptleistungen** → wähle eine Leistung → Text ändern → **Save**

### Referenzen
- Neues Projekt hinzufügen (Vorher/Nachher-Bilder + Text)
- Klicke auf **Referenzen** → bei "Projekte" auf **+** → Titel, Text, Datum und Bilder eingeben → **Save**

### Kontaktformular
- Texte des Formulars ändern (Überschrift, Button-Text, Meldungen)

---

## Nach dem Speichern

Die Website aktualisiert sich **automatisch in 1-2 Minuten**. Du musst nichts weiter tun.

---

## Wichtig

- **Du kannst nichts kaputt machen.** Wenn etwas schiefgeht, bleibt die alte Version online.
- **Bilder:** Am besten als JPG oder PNG hochladen (nicht zu groß, max. 2 MB).
- **Nicht ändern:** Design, Layout und Navigation — dafür bin ich da.
