## 5. Железные правила

| # | Правило | Причина |
|---|---------|---------|
| 1 | NO `next/image` → `<img>` + `getImageUrl()` | Website Mistakes #4: basePath bug |
| 2 | NO `var()` в `@theme inline` → direct hex only | PH Gotcha G1: breaks Tailwind v4 |
| 3 | NO hardcoded strings → all from `data/*.json` | PH правило: JSON data layer |
| 4 | NO stubs → deliverable = data on screen + test | PH Crisis #1: Team V1 fired |
| 5 | Honeypot: `left-[-9999px]`, NOT `display:none` | PH Gotcha G3: bots detect |
| 6 | Node 22 in CI | PH Gotcha G2: Node 24 unavailable |
| 7 | `.nojekyll` in deploy pipeline | PH Gotcha G6: _next/ not served |
| 8 | `force-static` in sitemap.ts/robots.ts | PH Gotcha G17: static export |
| 9 | `useSyncExternalStore` for matchMedia | PH Gotcha G4: React 19 |
| 10 | SplitText FOUC: `opacity:0` → set after split | PH Gotcha G9 |
| 11 | Sequential animations: single timeline, not separate triggers | PH Gotcha G10 |
| 12 | Floating labels: Tailwind `peer` + `placeholder=" "` | PH Gotcha G11 |
